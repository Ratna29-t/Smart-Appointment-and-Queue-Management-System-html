# Smart-Appointment-and-Queue-Management-System-html
Smart Appointment and Queue Management System

A responsive, UX-first **Smart Appointment & Queue Management System** designed to reduce wait-time frustration for users in clinics, banks, salons, and government offices. The interface focuses on clarity, live queue visibility, and accessibility.

> Tech stack: **HTML + CSS only** (no framework, no JS required to run)
